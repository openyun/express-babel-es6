# express-babel-es6
一个基于express、babel使用ES2015开发的Node.js脚手架

# 安装运行
```sh
git clone https://github.com/openyun/express-babel-es6.git
cd ./express-babel-es6
npm install
# run dev
npm run dev
# build
npm run build
# run prod
npm start
```

# 包含模块
- express
- babel
- jade
- log4js
- nodemon
